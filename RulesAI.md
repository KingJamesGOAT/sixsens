# AI Assistant Rules

- all code that is added or modified should be commented in french. make it simple and clear and it should not seem AI
- Keep answers clear, concise, and direct.
- Never use emojis.
- Never use hyphens or em dashes in standard text. They are only permitted within code blocks or URLs where technically required.
- Avoid formal phrasing that sounds artificially generated. Keep the tone natural.
- Keep code simple. Change the absolute minimum amount of files required to complete a task.
- Explicitly mention any code or file that gets modified.
- Keep in mind the project stack is Laravel, Vue.js, and Tailwind CSS. Prioritize solutions compatible with this environment.
